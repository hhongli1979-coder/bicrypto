# Futures v6.0.0

**Release Date:** December 2025
**Tags:** MAJOR RELEASE, COMPLETE REDESIGN, HIGH-RISK UI, ADMIN ENHANCEMENTS

## ⚠️ Important: V6 Compatibility

This is a **major version release** designed to work with Bicrypto Core v6.0.0. This update requires:

1. **Core v6.0.0 or higher** must be installed
2. **Complete backup** before upgrading
3. **Follow V5→V6 migration guide** if upgrading from V5
4. **Clear browser cache** after installation

## Overview

Version 6.0.0 introduces a revolutionary redesign of the Futures extension with an urgent, high-energy aesthetic. This release features an advanced amber and red color schema that embodies the high-risk, high-reward nature of futures trading, while delivering powerful admin tools and seamless integration with Bicrypto v6.

## Design System

### Urgency & High-Risk Color Schema

- **Primary Colors**: Amber & Red theme representing urgency and high-risk trading
  - **Amber**: Symbolizes caution, attention, and trading opportunities requiring quick action
  - **Red**: Represents urgency, high-risk positions, and critical market movements
- **High-Energy Aesthetics**: Dynamic visual language that conveys market urgency
- **Warning Gradients**: Bold gradients emphasizing risk and trading intensity
- **Dark Mode Intensity**: Dramatic dark mode with vibrant warning accents
- **Accessibility Standards**: WCAG 2.1 AA compliant with high contrast for critical alerts

### Visual Identity

- **Risk-Aware Design**: Amber accents emphasize caution and attention to risk
- **Urgency Indicators**: Red highlights convey critical actions and high-leverage positions
- **Professional Interface**: Intense, focused design aligned with V6 core styling
- **High-Stakes Feel**: Visual elements that communicate the serious nature of futures trading
