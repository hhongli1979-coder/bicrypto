‚ûï Creating a New Bot

1

Click "Create New Bot"

Click the "+" button in the bots management interface to open the creation dialog.

2

Select Market and Bot Type

Choose which AI Market this bot will trade on, and select its personality type.

3

Configure Trading Parameters

Set minimum/maximum trade amounts, risk tolerance, and other trading parameters.

4

Save and Activate

Save the bot configuration. The bot starts INACTIVE - activate it when the market is ready.

üìù Bot Configuration Fields

Basic Information

Bot Name
Text (Required)

Unique identifier for this bot. Use descriptive names like "ETH-Scalper-01" or "BTC-MarketMaker-Main".

AI Market
Select (Required)

The AI Market this bot will trade on. One bot can only trade on one market.

Bot Type
Select (Required)

The personality type that determines trading behavior:

SCALPER SWING MARKET_MAKER ACCUMULATOR DISTRIBUTOR
Status
Select

Current operational status:

ACTIVE INACTIVE PAUSED

Trading Parameters

Min Trade Amount
Decimal (Required)

Minimum amount per trade in base currency. Prevents dust trades.

Max Trade Amount
Decimal (Required)

Maximum amount per trade. Limits single-trade exposure.

Risk Tolerance
Decimal 0-1 (Required)

How aggressive the bot trades. 0 = conservative, 1 = aggressive. Affects trade size and frequency.

Base Balance
Decimal (Auto)

Bot's current balance in base currency. Tracked automatically.

Quote Balance
Decimal (Auto)

Bot's current balance in quote currency. Tracked automatically.

Total PnL
Decimal (Auto)

Cumulative profit/loss since bot creation. Updated after each trade.

Performance Tracking

Total Trades
Integer (Auto)

Total number of trades executed by this bot.

Last Trade At
DateTime (Auto)

Timestamp of the most recent trade executed by this bot.
