# FAQ v6.0.0

**Release Date:** December 2025
**Tags:** MAJOR RELEASE, COMPLETE REDESIGN, LANDING PAGE, TROUBLESHOOTER, UI/UX OVERHAUL, COLOR SCHEMA, ANIMATIONS

## ⚠️ Important: V6 Compatibility

This is a **major version release** designed to work with Bicrypto Core v6.0.0. This update requires:

1. **Core v6.0.0 or higher** must be installed
2. **Complete backup** before upgrading
3. **Follow V5→V6 migration guide** if upgrading from V5
4. **Clear browser cache** after installation

## Overview

Version 6.0.0 represents a transformative redesign of the FAQ extension with a clarity-focused aesthetic. This release introduces a premium landing page, innovative troubleshooter tool, and comprehensive redesign featuring sky blue color schema that embodies knowledge sharing and clarity, with enhanced animations and superior responsive design across all interfaces.

### Knowledge & Clarity Color Schema

- **Primary Colors**: Sky & Blue theme representing knowledge and clarity

- **Clear Aesthetics**: Clean, approachable visual language that encourages learning
- **Light Gradients**: Airy gradients creating spacious, welcoming atmosphere
