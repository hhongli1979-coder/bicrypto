# Ecommerce v6.0.0

**Release Date:** December 2025
**Tags:** MAJOR RELEASE, COMPLETE REDESIGN, LANDING PAGE, UI/UX OVERHAUL, COLOR SCHEMA, ANIMATIONS

## ⚠️ Important: V6 Compatibility

This is a **major version release** designed to work with Bicrypto Core v6.0.0. This update requires:

1. **Core v6.0.0 or higher** must be installed
2. **Complete backup** before upgrading
3. **Follow V5→V6 migration guide** if upgrading from V5
4. **Clear browser cache** after installation

## Overview

Version 6.0.0 represents a complete transformation of the Ecommerce extension with a warm, inviting aesthetic. This release introduces a brand-new landing page, unified amber and emerald color schema, and comprehensive redesign of all user and admin interfaces with enhanced animations, improved card designs, and superior responsive layouts.

## Design System

### Warm & Growth Color Schema

- **Primary Colors**: Amber & Emerald theme representing warmth and growth
  - **Amber**: Symbolizes warmth, energy, and vibrant commerce activity
  - **Emerald**: Represents growth, prosperity, and sustainable business success
- **Inviting Aesthetics**: Warm, welcoming visual language that encourages engagement
- **Natural Gradients**: Organic gradient combinations creating depth and vitality
- **Dark Mode Harmony**: Balanced colors that maintain warmth in dark mode
- **Accessibility Focused**: WCAG 2.1 AA standards with optimized color contrast

### Visual Identity

- **Commerce-Focused Design**: Amber accents emphasize products, sales, and transactions
- **Growth Indicators**: Emerald highlights showcase success metrics and analytics
- **Modern Marketplace**: Contemporary design aligned with V6 core styling
- **Emotional Connection**: Color psychology creates trust and shopping confidence
