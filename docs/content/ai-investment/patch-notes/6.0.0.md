# AI Investment v6.0.0

**Release Date:** December 2025
**Tags:** MAJOR RELEASE, COMPLETE REDESIGN, FUTURISTIC UI, COLOR SCHEMA

## ⚠️ Important: V6 Compatibility

This is a **major version release** designed to work with Bicrypto Core v6.0.0. This update requires:

1. **Core v6.0.0 or higher** must be installed
2. **Complete backup** before upgrading
3. **Follow V5→V6 migration guide** if upgrading from V5
4. **Clear browser cache** after installation

Version 6.0.0 introduces a revolutionary redesign of the AI Investment extension with a futuristic, innovative aesthetic. This release features a cutting-edge cyan and purple color schema that embodies the forward-thinking nature of AI-powered investment strategies, while delivering enhanced user experience and seamless integration with Bicrypto v6.
