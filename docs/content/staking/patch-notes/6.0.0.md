# Staking v6.0.0

**Release Date:** December 2025
**Tags:** MAJOR RELEASE, COMPLETE REDESIGN, LANDING PAGE, UI/UX OVERHAUL, COLOR SCHEMA, ANIMATIONS

## ⚠️ Important: V6 Compatibility

This is a **major version release** designed to work with Bicrypto Core v6.0.0. This update requires:

1. **Core v6.0.0 or higher** must be installed
2. **Complete backup** before upgrading
3. **Follow V5→V6 migration guide** if upgrading from V5
4. **Clear browser cache** after installation

## Overview

Version 6.0.0 represents a revolutionary transformation of the Staking extension with a premium, expertise-driven aesthetic. This release introduces a brand-new landing page, unified violet and indigo color schema, and comprehensive redesign of all staker and admin interfaces with enhanced animations, improved card designs, and superior responsive layouts that embody premium service, expertise, and deep trust.
