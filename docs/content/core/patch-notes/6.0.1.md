## API & Backend Changes

### New API Endpoints

- **User Media API**: `GET /api/user/media` - Blog authors can now fetch and manage their own media library
- **Email Wrapper API**: `GET /api/admin/system/notification/template/wrapper` - Fetch email wrapper for template preview
- **Landing Stats API**: `GET /api/content/landing-stats` - Fetch real-time platform statistics for landing page
- **NFT Landing Stats**: `GET /api/ext/nft/landing` - Fetch NFT marketplace statistics (extension-specific)

### API Key Management

- **Removed Payment Permission**: Eliminated deprecated "Payment" permission from API key creation and editing
- **Cleaner Permission List**: Streamlined API permission options to only include actively used permissions:
  - Trade: For placing orders and trading on the exchange
  - Futures: For trading in futures markets
  - Deposit: For viewing deposit addresses and history
  - Withdraw: For withdrawals from the account
  - Transfer: For transfers between accounts

### User Store

- **Updated API Permissions**: Removed obsolete payment API permission from user store configuration
- **Improved Type Safety**: Enhanced type definitions for API key management
