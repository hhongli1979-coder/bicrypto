## What's New

### 1. AI Market Maker Extension

Added a new powerful AI Market Maker extension that provides intelligent market making capabilities with AI-driven trading bots. This extension includes:

- **5 AI Bot Personalities**: SCALPER, SWING, MARKET_MAKER, ACCUMULATOR, and DISTRIBUTOR with unique trading behaviors
- **Advanced Price Strategies**: Gradual drift, oscillation, and support/resistance price movement patterns
- **Hybrid Liquidity System**: Control the split between simulated AI orders and real ecosystem orders
- **Risk Management**: Built-in safety features including volatility pause, loss limits, and circuit breakers
- **Market Analytics**: Comprehensive dashboard for monitoring AI trading performance

For detailed installation and configuration instructions, please refer to the AI Market Maker documentation in the extensions section.

### 2. NFT Extension Support

Added comprehensive support for NFT extension functionality including:

- NFT marketplace and auction systems
- Collection and creator management
- NFT categories and reviews
- Royalty distribution system
- Dispute resolution system
- Fractional NFT ownership

### 3. Translation System Enhancement

Comprehensive translation update across all supported locales:

- Updated all existing translations with latest content
- Added missing translation keys across all 100+ languages
- Improved translation consistency and accuracy
- Complete coverage for all UI elements

## Migration Notes

### AI Market Maker Addon

**IMPORTANT**: If you are updating from a previous version of the AI Market Maker addon, you must delete the following file:

**File to delete:**

```plaintext
frontend/app/[locale]/(ext)/admin/ai/layout.tsx
```

**Steps to perform migration:**

1. Navigate to your project's frontend directory
2. Delete the file at `frontend/app/[locale]/(ext)/admin/ai/layout.tsx`
3. Run `pnpm updator` to complete the migration
