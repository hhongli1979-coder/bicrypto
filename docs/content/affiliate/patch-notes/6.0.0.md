# Affiliate v6.0.0

**Release Date:** December 2025
**Tags:** MAJOR RELEASE, COMPLETE REDESIGN, LANDING PAGE, UI/UX OVERHAUL, COLOR SCHEMA

## ⚠️ Important: V6 Compatibility

This is a **major version release** designed to work with Bicrypto Core v6.0.0. This update requires:

1. **Core v6.0.0 or higher** must be installed
2. **Complete backup** before upgrading
3. **Follow V5→V6 migration guide** if upgrading from V5
4. **Clear browser cache** after installation

## Overview

Version 6.0.0 represents a complete visual and functional redesign of the Affiliate extension. This release introduces a brand-new landing page, unified color schema, and enhanced user experience across all pages, perfectly integrated with the new Bicrypto v6 design language.

## Design System

### Brand Color Schema

- **Primary Colors**: Blue & Amber theme representing trust and rewards
  - **Blue**: Represents trust, reliability, and professional network building
  - **Amber**: Symbolizes rewards, earnings, and achievement
- **Consistent Application**: Color schema applied uniformly across all components
- **Dark Mode Support**: Full dark mode compatibility with optimized color contrast
- **Accessibility Compliant**: WCAG 2.1 AA standards for color contrast and readability

### Visual Identity

- **Trust-Based Design**: Blue accents emphasize network reliability and platform security
- **Reward-Focused Elements**: Amber highlights draw attention to earnings, commissions, and achievements
- **Modern Aesthetics**: Clean, contemporary design aligned with V6 core styling
- **Visual Hierarchy**: Improved information architecture for better user comprehension
