What is the Payment Gateway Extension?

The Payment Gateway extension transforms your Bicrypto platform into a complete cryptocurrency payment processor. It allows external websites (like WooCommerce, Shopify, or custom applications) to accept cryptocurrency payments from their customers, with funds being processed through your platform's wallet system.

Merchants can register on your platform, create API keys, and integrate payment processing into their websites. Customers pay using any wallet type (FIAT, SPOT, or ECO) with automatic currency conversion. The system handles fee collection, refunds, payouts, and provides comprehensive dashboards for both merchants and administrators.

The extension includes a WooCommerce plugin for easy WordPress integration, customizable checkout page designs, webhook notifications, and test mode for safe development.

Initial Configuration

Configure Gateway Settings

Set up basic gateway parameters and fee structures:

  1. 1. Go to Admin ‚Üí Gateway ‚Üí Settings
  2. 2. Configure fee percentage (platform fee on each transaction)
  3. 3. Set minimum payout threshold
  4. 4. Configure payout schedule (daily, weekly, monthly)
  5. 5. Enable/disable wallet types (FIAT, SPOT, ECO)
  6. 6. Configure allowed currencies for payments

Set Up Payout Cron Job

Enable automated payout processing:

  1. 1. Navigate to Admin ‚Üí System ‚Üí Cron Jobs
  2. 2. Find the "Gateway Payout" cron job
  3. 3. Enable the cron job
  4. 4. Configure the schedule based on your payout frequency

The cron automatically checks merchant gateway balances and creates payout requests for eligible merchants.

Configure User Permissions

Set up roles and access levels for gateway management:

  1. 1. Go to Admin ‚Üí Roles & Permissions
  2. 2. Create or edit roles for gateway access
  3. 3. Assign merchant management permissions
  4. 4. Set payout approval permissions
  5. 5. Configure refund processing permissions

Payment Flow Overview

1

Payment Intent Created

External site calls the API to create a payment intent with amount, currency, and order details

2

Customer Redirected to Checkout

Customer is redirected to your platform's checkout page to complete payment

3

Customer Selects Wallet

Customer chooses which wallet(s) to pay from, with automatic currency conversion

4

Payment Processed

Funds debited from customer wallet, fees collected, merchant gateway balance updated

5

Webhook Notification

Payment confirmation sent to merchant's webhook URL with payment details

6

Payout Processing

Admin approves payout, funds move from gateway balance to merchant's wallet

WooCommerce Integration

The extension includes a ready-to-use WooCommerce plugin for WordPress sites:

Installation Steps

  1. Download the plugin from your Gateway dashboard
  2. Upload to your WordPress site via Plugins > Add New > Upload Plugin
  3. Activate the plugin
  4. Go to WooCommerce > Settings > Payments
  5. Enable "Bicrypto Payment Gateway"
  6. Enter your API URL, Public Key, and Secret Key
  7. Configure webhook URL in your Bicrypto Gateway settings

The plugin automatically handles payment creation, status updates via webhooks, and order status synchronization.
