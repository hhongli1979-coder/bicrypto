# Mailwizard v6.0.0

**Release Date:** December 2025
**Tags:** MAJOR RELEASE, COMPLETE REDESIGN, ADMIN ENHANCEMENTS, PROFESSIONAL UI

## ⚠️ Important: V6 Compatibility

This is a **major version release** designed to work with Bicrypto Core v6.0.0. This update requires:

1. **Core v6.0.0 or higher** must be installed
2. **Complete backup** before upgrading
3. **Follow V5→V6 migration guide** if upgrading from V5
4. **Clear browser cache** after installation

## Overview

Version 6.0.0 introduces a revolutionary redesign of the Mailwizard extension with a professional communication aesthetic. This release features an advanced violet and rose color schema that embodies professional email marketing and elegant communication design, while delivering powerful admin tools and seamless integration with Bicrypto v6.
