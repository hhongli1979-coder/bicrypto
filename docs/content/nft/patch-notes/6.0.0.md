# NFT v6.0.0

**Release Date:** December 2025
**Tags:** MAJOR RELEASE, COMPLETE REDESIGN, ADMIN ENHANCEMENTS, CREATIVE UI

## ⚠️ Important: V6 Compatibility

This is a **major version release** designed to work with Bicrypto Core v6.0.0. This update requires:

1. **Core v6.0.0 or higher** must be installed
2. **Complete backup** before upgrading
3. **Follow V5→V6 migration guide** if upgrading from V5
4. **Clear browser cache** after installation

## Overview

Version 6.0.0 introduces a revolutionary redesign of the NFT extension with a creative, artistic aesthetic. This release features an advanced purple and pink color schema that embodies digital creativity and contemporary art culture, while delivering powerful admin tools and seamless integration with Bicrypto v6.
